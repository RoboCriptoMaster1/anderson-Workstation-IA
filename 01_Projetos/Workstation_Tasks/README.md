# 🚀 Workstation Tasks

Sistema de gerenciamento de projetos e tarefas desenvolvido como o primeiro projeto oficial da **Workstation IA**.

---

# Objetivo

O Workstation Tasks tem como objetivo organizar o desenvolvimento da Workstation IA e, futuramente, gerenciar qualquer projeto de software de forma organizada, modular e escalável.

---

# Instalação

Clone o repositório:

```bash
git clone <repositorio>
```

Acesse a pasta do projeto:

```bash
cd Workstation_Tasks
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

---

# Execução

Execute a aplicação:

```bash
python app.py
```

Acesse no navegador:

```text
http://127.0.0.1:5000
```
