# 🚀 Workstation IA

Ambiente profissional para desenvolvimento de software, Inteligência Artificial, automação, análise de dados e criação de agentes de IA.

A **Workstation IA** é uma plataforma pessoal desenvolvida para centralizar projetos, bibliotecas, integrações, APIs, automações, banco de dados e documentação técnica em uma única estrutura organizada, reutilizável e escalável.

---

# Objetivo

A Workstation IA foi criada para servir como plataforma principal de desenvolvimento de soluções tecnológicas.

Seu objetivo é permitir a criação de:

- Sistemas Web
- APIs
- Agentes de Inteligência Artificial
- Aplicações SaaS
- Automações
- Dashboards
- Banco de Dados
- Ferramentas em Python
- Ferramentas em Node.js
- Soluções para Excel
- Documentação Técnica
- Bibliotecas reutilizáveis

Todo projeto desenvolvido deverá seguir a estrutura e os padrões definidos nesta Workstation.

---

# Estrutura Oficial

O trabalho é organizado por categorias.

Sempre respeite esta organização ao criar novos projetos.

| Pasta | Finalidade |
|--------|------------|
| `01_Projetos` | Projetos completos |
| `02_Agentes` | Agentes de IA |
| `03_APIs` | Integrações e APIs |
| `04_Banco_Dados` | Bancos de Dados, SQL, MER e DER |
| `05_Excel` | Excel, VBA, Office Scripts e Power Query |
| `06_Python` | Scripts e projetos em Python |
| `07_NodeJS` | Projetos em Node.js |
| `08_Markdown` | Documentação em Markdown |
| `09_Documentos` | Documentos gerais |
| `10_Templates` | Templates reutilizáveis |
| `11_Prompts` | Biblioteca de Prompts |
| `12_Automacoes` | Scripts e Fluxos de Automação |
| `13_MCP` | Servidores e configurações MCP |
| `14_Modelos_Locais` | Modelos locais de IA |
| `15_Config` | Configurações da Workstation IA |
| `99_Backup` | Backups |

As pastas vazias são mantidas no Git através de arquivos `.gitkeep`.

# Como começar

## 1. Clonar o repositório

```bash
git clone https://github.com/RoboCriptoMaster1/anderson-Workstation-IA
```

---

## 2. Abrir no VS Code

Abra a pasta **Workstation-IA** utilizando o Visual Studio Code.

---

## 3. Verificar a estrutura

Execute:

```powershell
python .\06_Python\verificar_estrutura.py
```

Se todas as pastas estiverem corretas será exibido:

```text
Resultado: estrutura 100% correta!
```
