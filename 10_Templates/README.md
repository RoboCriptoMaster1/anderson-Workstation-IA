# 📁 Templates

## Objetivo

Esta pasta contém modelos oficiais utilizados pela Workstation IA.

Os templates têm como objetivo acelerar o desenvolvimento de novos projetos, mantendo padronização, organização e reutilização de código.

---

# Organização

Cada subpasta contém modelos específicos para uma tecnologia.

| Pasta | Finalidade |
|--------|------------|
| Python | Templates para projetos Python |
| NodeJS | Templates para aplicações Node.js |
| HTML | Estrutura inicial de páginas HTML |
| CSS | Arquivos de estilo reutilizáveis |
| JavaScript | Scripts JavaScript |
| SQL | Scripts de banco de dados |
| Excel | Modelos para Excel, VBA e Office Scripts |
| Markdown | Modelos de documentação |
| Git | Arquivos padrão de repositórios Git |
| Docker | Templates para containers |
| SaaS | Estrutura inicial para aplicações SaaS |

---

# Regras

Todos os templates devem:

- seguir o padrão da Workstation IA;
- conter comentários quando necessário;
- ser reutilizáveis;
- possuir documentação quando aplicável;
- evitar duplicação de código.

---

# Evolução

Novos templates poderão ser adicionados conforme a evolução da Workstation IA.
