# Objetivo

Esta pasta concentra todos os prompts oficiais utilizados pela Workstation IA.

# Padrão dos Prompts

Todo prompt deverá conter, sempre que possível:

- Objetivo
- Contexto
- Instruções
- Restrições
- Resultado esperado

---

# Convenções

Os nomes dos arquivos devem utilizar:

- letras minúsculas
- underscore (_)
- nomes descritivos

Exemplos:

```text
criar_api_python.md

analisar_sql.md

gerar_dashboard_powerbi.md
```
