# Missão

O Cortex é o cérebro oficial da Workstation IA.

Toda inteligência da plataforma é construída, organizada e mantida dentro do Cortex.

O Cortex centraliza:

- conhecimento técnico
- arquitetura
- memória do projeto
- documentação
- padrões
- decisões técnicas
- fluxos
- comandos
- prompts
- integrações
- componentes
- contexto

O Claude Code é utilizado como tecnologia de apoio para raciocínio, desenvolvimento e automação.

O Claude Code **não substitui o Cortex**.

O conhecimento pertence ao Cortex.

---

# Objetivo

Construir uma Base de Conhecimento modular, versionada e escalável capaz de alimentar toda a inteligência da Workstation IA.

Toda informação utilizada pelo sistema deverá possuir uma única fonte oficial dentro desta Base de Conhecimento.

---

# Filosofia

O Cortex segue os seguintes princípios.

- Arquitetura Limpa
- Simplicidade
- Modularidade
- Baixo Acoplamento
- Alta Coesão
- Escalabilidade
- Reutilização
- Documentação Contínua
- Versionamento
- Evolução Incremental

---

# Fluxo Oficial

Todo conhecimento seguirá obrigatoriamente o seguinte fluxo.

```text
Pesquisa

↓

Validação

↓

Knowledge Base

↓

Indexação

↓

CLAUDE.md

↓

Cortex

↓

Workstation IA
```

O arquivo `CLAUDE.md` será sempre gerado a partir da Base de Conhecimento.

Nenhuma informação deverá ser adicionada diretamente ao `CLAUDE.md`.
