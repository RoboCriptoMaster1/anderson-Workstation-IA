# Referências

## Objetivo

Centralizar todas as referências oficiais utilizadas pelo Cortex e pela Workstation IA.

Este documento representa o catálogo oficial de tecnologias, documentações, normas, especificações e materiais técnicos homologados para consulta.

Nenhuma referência externa deverá ser utilizada como padrão sem estar registrada neste documento.

---

# Filosofia

O conhecimento do Cortex deverá ser verificável.

Toda decisão técnica deverá possuir uma fonte oficial.

A documentação oficial sempre terá prioridade sobre artigos, blogs ou conteúdos de terceiros.

# Critérios de Homologação

Uma nova referência somente poderá ser adicionada quando.

- possuir documentação oficial;
- for amplamente utilizada;
- possuir estabilidade;
- agregar valor à arquitetura;
- não contrariar os padrões do Cortex.
